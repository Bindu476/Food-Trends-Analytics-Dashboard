## 🌟 About The Project

Food Trends Analytics Dashboard is a Business Intelligence solution built using **Power BI** to uncover customer food preferences, nutrition patterns, and social media influence on food trends.

This project transforms raw food-related data into interactive dashboards that help businesses understand customer behavior, identify popular food categories, monitor engagement, and make informed strategic decisions.

---

## 🎯 Problem Statement

Food businesses generate large volumes of customer and product data, but extracting meaningful insights can be challenging.

This dashboard helps answer key business questions:

✅ Which food categories are most popular?

✅ What nutritional patterns influence customer choices?

✅ Which products receive the highest ratings?

✅ How does social media impact food popularity?

✅ What trends can businesses leverage for growth?
